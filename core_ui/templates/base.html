What is RAG

RAG (Retrieval-Augmented Generation) is a mode where model responses are backed by search through your knowledge base. Upload documents to Knowledge Base, enable "RAG Enabled" in chat, and responses will reference those documents. Great for internal policies, company knowledge bases, and documentation.

What is Workflow

Workflow (Orchestrator) is a chain: input, analysis, tool actions, response. The agent plans steps, calls tools (files, SSH, web) and forms a final answer.

How to choose a model

Currently only Cursor Auto is used. In Settings (Models) you set the default provider and models for chat, agent, and RAG; Gemini and Grok will be available later.

How to safely connect a server

Server connections are configured in the Servers section. Use keys (SSH) and credentials from the Passwords section — do not enter passwords in chat. Limit user permissions on the server and store keys only in secure storage (.env, password manager), not in code or chat.
